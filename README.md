# 🦄 Mundo da Yoyo

Site de jogos educativos feito com carinho para a Yoyo.

## 📂 O que tem aqui

Este projeto é um site estático com jogos infantis:

- **Palavras** (`silabas.html`) — junte sílabas e forme palavras
- **Números** (`numeros.html`) — conte os objetos e escolha o número
- **Formar Números** (`formar-numeros.html`) — monte números e ouça a pronúncia
- **Alfabeto** (`alfabeto.html`) — monte palavras letra por letra
- **Frases** (`frases.html`) — monte frases com palavras soltas
- **Memória** (`memorias.html`) — encontre os pares iguais
- **Mundo Mágico** (`magical.html`) — palavras especiais de unicórnios e arco-íris

## 🚀 Como executar

Como é um site estático, basta abrir o arquivo `index.html` no navegador:

1. Abra a pasta do projeto.
2. Dê dois cliques no arquivo `index.html`.
3. O site abrirá no navegador padrão.

> Não precisa instalar nada — funciona no Chrome, Edge, Firefox ou qualquer navegador moderno.

## 💻 Como copiar para outro computador (notebook da Yohana)

### Opção 1 — Pendrive ou nuvem (mais simples)
Copie toda a pasta `MundoDaYoyo` para o outro computador e abra o `index.html`.

### Opção 2 — GitHub (para atualizar facilmente no futuro)
Se quiser enviar para um repositório online:

1. Crie um repositório no GitHub.
2. No terminal, dentro desta pasta, execute:
   ```bash
   git remote add origin https://github.com/SEU_USUARIO/NOME_DO_REPO.git
   git branch -M main
   git push -u origin main
   ```
3. No notebook da Yohana, abra o terminal e execute:
   ```bash
   git clone https://github.com/SEU_USUARIO/NOME_DO_REPO.git
   ```
